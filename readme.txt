Данный обработчик необходим для распределения всех файлов MT для входящих ПТ и исходящих ПТ

скрипты move_file_by_content_to_b_halyk.bat и move_file_by_content_to_c_halyk.bat


Дано:

Входящая папка А. Содержит все файлы из Фасти банка.


Папка B, в которую попадают файлы по системе, которая обрабатывает входящие требования:

PTP(MT100)
EAR
RZAP
OP1

Скрипт move_file_by_content_to_b_halyk.bat перекладывает все файлы, содержащие вышеуказанные форматы в папку B

------------------


Папка С, в которую попадают файлы по системе, которая обрабатывает файлы исходящих требований:


PR1
PC6
EAC
PC3
PC2

Скрипт move_file_by_content_to_c_halyk.bat перекладывает все файлы, содержащие вышеуказанные форматы в папку C
